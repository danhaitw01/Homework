關於段位

段位內含一組四首的指定難度歌曲，血條共用於整組曲子。

段位可以消耗體力或是次元結晶進行遊玩。

以下是完整段位名稱列表。

  • Phase 1：新世界的第一步
  • Phase 2：一次心跳的加速
  • Phase 3：永不屈服的精神
  • Phase 4：光芒的永恆之地
  • Phase 5：玻璃之中的殘酷
  • Phase 6：沉浸於悲喜交加
  • Phase 7：在命運到來之際
  • Phase 8：趨於混沌的時空
  • Phase 9：分崩離析的自我
  • Phase 10：光與對立的洪流
  • Phase 11：創世之曙光
